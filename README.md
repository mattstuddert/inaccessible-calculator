# Inaccessible Calculator

This repository is a deliberately flawed implementation of the [Frontend Mentor Calculator app challenge](https://www.frontendmentor.io/challenges/calculator-app-9lteq5N29). It is used internally at [Frontend Mentor](https://www.frontendmentor.io) as a test project for our automated testing and AI code review tooling.

The code is intentionally written with realistic beginner-level mistakes across HTML, CSS, and JavaScript, including accessibility issues, semantic problems, and common JavaScript anti-patterns. It is **not** an example of how to build a calculator and should not be used as a reference implementation.

## Why this exists

We use this project to:

- Validate the kinds of issues our automated code review surfaces.
- Test how feedback is presented to learners on the platform.
- Iterate on the prompts and rules behind our review tooling.

If you've stumbled across this repo from a Frontend Mentor announcement and want to poke around, feel free. Just keep in mind that the goal here is to provide a noisy, imperfect codebase for our tools to review — not to ship a polished calculator.

## Project structure

- `index.html`, `styles.css`, `script.js` — the calculator implementation.
- `design/` — desktop and mobile design references for the original challenge.
- `images/` — challenge assets (favicon, etc.).
- `style-guide.md` — colors, typography, and layout specs from the challenge.
- `preview.jpg` — the design preview image from the original challenge.
